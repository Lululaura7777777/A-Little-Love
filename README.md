# A Little Love ❤️

A tiny interactive webpage made for someone special.

✨ Built with a simple idea:  
sometimes the smallest things can make someone smile.

---

## Features

- 🌌 Animated starry night background  
- ❤️ Falling hearts animation  
- 💖 Click anywhere to release hearts  
- ✨ Soft glow and romantic visual effects  
- 📱 Works on both desktop and mobile

---

## Live Demo

Open the page here:

https://lululaura7777777.github.io/A-Little-Love/

---

## How It Works

The page is built using a single `index.html` file with:

- HTML
- CSS animations
- JavaScript for interactive effects

When the page loads:

1. A starry background appears
2. Hearts begin to fall across the screen
3. Clicking anywhere creates a burst of hearts and sparkles
